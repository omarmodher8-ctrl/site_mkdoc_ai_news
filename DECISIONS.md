# DECISIONS

## 2025-09-18 - 依存パッケージのバージョン固定を行わない
- `requirements.txt` はバージョン未指定のまま運用する。
- Cloudflare Pages のビルドで異常が起これば検知できるため、その時点で対処する。
- 必要になった場合のみ、直前に動作したバージョンをメモしつつ段階的に固定する方針。

## 2025-09-18 - `.memo` ファイルの二重読み込みを維持
- プラグインとマクロの両方で `.memo` を読む冗長構成はそのままにする。
- 冗長でも安全側に倒しておきたいという意図で、パフォーマンス影響は小さい。

## 2025-09-18 - カテゴリ表示名のハードコードをフォールバックとして残す
- `macros/home.py` の `CATEGORY_BASE_LABELS` はナビゲーションのタイトル取得が失敗した際の保険とする。
- `.pages` でタイトルを定義できるが、ナビゲーション情報が得られないケースに備えて残置。
- コメントで意図を明記済み。
