# AI関連情報まとめ (2026-03-05)

## 1. 📊 今日の概要 (Today's Overview)
本日のAI業界は、AIが「指示を待つツール」から「自ら実行するパートナー（エージェント）」へと進化を遂げた記念すべき一日となりました。OpenAIが「エージェントモード」を搭載したGPT-5.2をリリースし、実務自動化の基準を塗り替える一方で、YouTubeはクリエイター向けに生成AI動画編集機能「Add object」「Reimagine」を公開。AIが日常生活やプロフェッショナルな制作現場のワークフローに、より深く、かつ自律的に統合され始めています。

- **GPT-5.2の衝撃**: 自律的にブラウザを操作し、購入や手続きまで代行する「エージェントモード」が実装。
- **YouTube Shortsの進化**: プロンプトだけで動画内に物を追加したり、動画のスタイルを全置換する機能がテスト開始。
- **AI Slop（AIのゴミ）対策**: YouTube CEOが低品質なAI量産コンテンツの排除を強化すると明言。
- **Google Nano Banana 2**: 画像内のテキスト描写能力を極限まで高めた最新画像生成モデルが登場。

## 2. 🔥 注目AIツール・サービス ランキング (Trending AI Tools & Services)

1. **GPT-5.2 (OpenAI)**
   - **概要**: 「エージェントモード」を初搭載した次世代大規模言語モデル。
   - **何ができるか**: ユーザーに代わってWebサイトを閲覧し、データの抽出、フォーム入力、さらには決済までを自律的に完結。
   - **注目理由**: 「AIに相談する」フェーズから「AIに任せる」フェーズへの決定的な転換点となったため。

2. **YouTube Shorts AI (Add object / Reimagine)**
   - **概要**: Shortsのリミックス機能に統合された生成AI動画編集ツール。
   - **何ができるか**: 撮影済みの動画に「猫を追加して」と入力したり、動画全体を「油絵風のスタイル」に変身させることが可能。
   - **注目理由**: 高度な動画編集スキルがなくても、プロフェッショナルなVFX効果をスマホ一台で実現できるため。

3. **Google Nano Banana 2**
   - **概要**: Googleの最新画像生成AIモデル。
   - **何ができるか**: 画像内の看板や商品のラベル、手書き文字などを極めて正確に生成・描写。
   - **注目理由**: これまで生成AIが苦手としていた「画像内テキスト」の破綻を克服し、広告デザイン等への実用性が飛躍的に向上したため。

4. **Thunderbit**
   - **概要**: ブラウザ上で動作するAI自動化拡張ツール。
   - **何ができるか**: 複数のWebサイトからの情報収集や、スプレッドシートへの転記を自然言語の指示だけで自動化。
   - **注目理由**: プログラミング知識なしで、日常のルーチン業務を「エージェント」化できる手軽さが話題。

5. **Granola**
   - **概要**: 文脈理解に特化したAI議事録・メモツール。
   - **何ができるか**: オンライン会議の音声を解析し、単なる文字起こしではなく、決定事項や次のアクションを「人間が書いたように」整理。
   - **注目理由**: 議事録作成という「最も退屈な作業」を完璧に自動化し、生産性を直感的に向上させるため。

## 3. 📈 トレンド分析 (Trend Analysis)

### 技術的トレンド：AIエージェントによる「自律実行」の確立
これまでのAIはプロンプト（指示）に対して「回答」を生成するだけでしたが、GPT-5.2に代表される最新モデルは、自らブラウザやツールを操作して「タスクを完結させる」能力（エージェント能力）を標準搭載し始めました。これにより、AIは情報検索の道具から、実作業の代行者へと技術的な次元が一段階上昇しました。

### 応用面のトレンド：GEO（生成エンジン最適化）へのシフト
従来の検索エンジン最適化（SEO）に代わり、AIが生成する回答（PerplexityやGoogle Search Generative Experience、GPT-5.2の検索モードなど）に自社の情報が採用されるための最適化「GEO（Generative Engine Optimization）」が、マーケティング業界の最重要課題となっています。
