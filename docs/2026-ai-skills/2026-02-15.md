### OpenAI CodexのSkills関連の注目点
Codexは2026年2月頃に大きなアップデートがあり、**Skills**がエージェントの効率を劇的に向上させるキー機能として話題に。Skillsは再利用可能な指示・テンプレート・API構成のセットで、複雑タスクを確実に実行可能。

- **CodexアプリのSkillsライブラリ**：Codexアプリ（macOS専用、2026年2月2日リリース）でSkillsをブラウズ・作成・管理できるUIが搭載。例として「design-to-code」スキル（FigmaデザインをプロダクションUIコードに変換）や「project management」スキル（Linearなどのツールでバグトリアージ・チケット追跡）が挙げられる。エージェントが並行ワークフローを扱い、バックグラウンドで長時間タスクを実行可能。[@kr0der](https://x.com/kr0der/status/2022806704170139820) はCodexのTipsとして、Skillsのfrontmatterをしっかり書くと自動呼び出しが優秀で、手動呼び出しが減ると指摘。Yoloモード（プロンプトインジェクション注意）や/reviewコマンドとの組み合わせでバグを最小化できる。

- **AgentSysツールの対応**：開発者[@avi_fenesh](https://x.com/avi_fenesh/status/2022778561191358821) が構築中のAgentSys（モジュラーランタイム）はCodex CLI対応で、27個のSkillsを含む41エージェント・12プラグインをサポート。agentic coding時代向けのツールとして注目。

- **Tips共有**：Codexは計画モード（plan mode）が優秀で、長タスクに強く、frontend以外ではOpusより優位との声も。Skillsを活用するとエージェントが経験から自動で呼び出すため、フロントマターの質が重要。

### Gemini関連のSkills/Agent機能
Geminiはコーディング特化のアップデートが続き、Skillsやエージェント開発キットが企業向けに強化。

- **GEARプログラムとAgent Development Kit (ADK)**：Googleが2026年2月11日に公開した**Gemini Enterprise Agent Ready (GEAR)**プログラムで、企業級エージェント構築を支援。ADK（オープンソース）を使ったSkills/ワークフローの学習パスを提供。Gemini Code Assistと統合し、reasoning loopsで予測可能な結果を出力。無料クレジット付きで即コード実装可能。

- **Gemini CLIのSkills/Hooks/Plan Mode**：Gemini CLIでSkillsを活用したagentic機能が強化。計画モードやhooksで複雑タスクを扱い、コーディングの「vibe coding」モデルとして高評価。[@shanegJP](https://x.com/shanegJP/status/2022693592276603266) はGeminiの多言語理解・プレゼンスキルが最高クラスで、実世界課題解決に期待と発言。
