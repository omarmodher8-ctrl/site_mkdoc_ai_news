**過去1週間（2026年5月3日〜10日頃）のAIコーディングツール（主にGemini、Codex、Cursor/CodeCode関連）の「Skills」に関する主な動きをまとめます。** Skillsとは、AIエージェント（Claude Code、Codex、Gemini CLI、Cursorなど）に特定のタスク・ベストプラクティス・ワークフローを教える再利用可能なMarkdownベースの指示セット（SKILL.mdなど）で、Agent Skills仕様に従い、ツール間で互換性が高まっています。

### 1. GitHub上の注目リポジトリとSkillsコレクション
- **VoltAgent/awesome-agent-skills**：公式開発チームとコミュニティによる1,000以上のAgent Skillsをキュレーションしたリポジトリ。Claude Code、Codex、Gemini CLI、Cursorなど幅広いツールに対応。5月8日頃にXquik agent skillなどが追加され、活発に更新中。公式スキル例としてGoogle Geminiのgemini-interactions-api（テキスト生成、チャット、マルチモーダル、関数呼び出しなど）、Stripeのベストプラクティス、HashiCorpのTerraform関連などが収録。

- **affaan-m/everything-claude-code**：Claude Code/Codex/Cursorなどを強化するオールインワンフレームワーク。Skills、memory、security、research-first開発を統合。GitHubで17万スター超の人気リポジトリとして言及され、.skills/ディレクトリなどで実践的なスキルセットを提供。

これらのリポジトリは、AIエージェントに「エンジン（モデル）だけでなく、仕事のやり方を教える」ためのもので、インストールして即利用可能（例: gh skill install など）。

### 2. Codex関連のSkills/機能アップデート
Codex（OpenAIのAIコーディングパートナー）ではSkillsがコード作成を超えた貢献（コード理解、プロトタイピング、PR化など）を可能にしています。

- 5月8日頃のCodex 0.130.0リリース：remote-control（headless app-server）、plugin detailsのbundled hooks強化、plugin sharingのmetadataなど。Skills/プラグインの扱いが改善され、Claude Codeからの移行もワンクリックでツール設定・指示・Skills・プロジェクト・過去セッションを移行可能に。速度の良さが評価されています。

ユーザーは「Codex+skillsで多くのタスクを完結できる」との実践報告あり。

### 3. Gemini関連のSkills/CLI強化
- Gemini CLIやGemini API Skills：gemini-interactions-apiスキルなどで最新APIの使い方（テキスト/チャット/ストリーミング/関数呼び出し/構造化出力）を教えるもの。Googleのagents-cliもSkillsを注入してエージェント構築を支援。

- Gemini CLIで過去セッションからAgent Skillsを自動作成する機能が便利と指摘（繰り返しタスク用）。

- Chrome版GeminiでSkills追加：チャット履歴からプロンプトをSkill保存し、/コマンドで呼び出し（ブラウザ操作、研究、タブ横断など）。確認が必要な敏感アクションもあり。

### 4. Cursorや共通のSkills活用事例
Cursor（VS Codeフォーク）では.skills/フォルダにMarkdownでドメイン特化ワークフローや規約を置き、動的にロード。Find Skillsツールで数百のSkillから最適なものを自動提案する使い方が日本ユーザー経由で海外バズ。

Skillsはplanning→implementation→review→learningのワークフローを分解・合成可能で、複数のAIツール（Claude Code/Codex/Cursor/Gemini CLI）で再利用しやすい点が強み。

### 5. その他の傾向
- Agent Skills仕様の普及により、GitHub CLIでSkills管理が可能に。セキュリティ監査やsupply chain対策の議論も。
- マーケティング/プロダクト/セキュリティなどコーディング以外のSkillsも充実（awesome-agent-skills参照）。

**まとめると**、今週はSkillsの「自動発見・作成・共有・クロスツール互換」が進み、単なるプロンプト集から実践的な「エージェントの職業訓練」へと成熟。GitHubのawesomeリストやeverything-claude-codeから始め、Codex/Geminiの最新アップデートを組み合わせるのがおすすめです。情報はXやGitHubのリアルタイム更新に基づき、古いものは除外しています。詳細は各リポジトリやリリースノートで確認を。
