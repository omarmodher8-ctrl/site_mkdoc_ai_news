### OpenAI CodexのSkills
OpenAIのCodexアプリでは、「**Skills**」がライブラリ形式で提供され、数百種類のツール・ワークフローをパッケージ化可能になりました。これにより、従来定義しにくかったタスクを再利用可能な能力として拡張できます。

具体的な人気Skills例：
- **Implement designs**：Figmaからデザインコンテキスト・資産・スクリーンショットを取得し、1:1の視覚一致で本番用UIコードに変換。
- **Manage projects**：Linearでバグトリアージ、リリース追跡、チームワークロード管理などを実施。
- その他：evals実行、トレーニングラン監視、ドキュメント作成、成長実験レポートなど。

これらのSkillsはアプリ内で作成・インストール可能で、CLIやIDEにも共有できます。Codexアプリのmulti-agent機能と組み合わせ、並行タスクやworktreesでの分離実行が強化されています。X上では、Codexを「manager/director型」としてplanningやscaffoldingに使い、Skillsで自動化を拡張するワークフローが話題です。

Codex CLIでは、cloud task delegation（`codex cloud exec`）でクラウド環境をスピンアップし、エージェントにタスクを委譲する機能もSkills拡張と連動して注目されています。1Mトークンコンテキストを活かした長時間タスク処理が強みです。

### Gemini（Google）のAgent Skillsと関連機能
Googleは4月1日頃に**Gemini API Docs MCP**と**Gemini API Developer Skills**を公開し、コーディングエージェントの性能向上を図っています。

- **Gemini API Docs MCP**：最新のGemini APIドキュメント、SDK、モデル情報をエージェントにリアルタイム接続（Model Context Protocol経由）。訓練データのカットオフによる古いコード生成を防ぎます。
- **Gemini API Developer Skills**：ベストプラクティス指示、リソースリンク、パターンを追加。SDKの最新パターンへ誘導します。

組み合わせにより、evalセットで96.3%のパス率、トークン使用量を63%削減する効果が確認されています。Gemini CLIでは、ci skill（自動失敗再現）、github-issue-creator skill、重複コード検出skillなどの追加がリリースノートで報告されています。

Google ColabのGemini統合では、**Learn Mode**（パーソナライズドAIコーディングチュータ）とCustom Instructionsが強化され、ステップバイステップ学習支援が向上。Agent ModeではGemma 4のローカル実行でAndroid開発向けのrefactoringやアプリ構築が可能です。

X上では、Gemini 3.1 Proのagentic capabilitiesやGoogle ecosystem統合が「architect/principal engineer型」として評価されており、複雑問題解決やセキュリティ脆弱性特定に強いとされています。

### CodeCode / Claude Code系のSkills
「CodeCode」は主にClaude Code（AnthropicのCLIベースエージェント）を指す文脈で登場し、Skills/Pluginsとして拡張されています。GitHub上でコミュニティ主導のSkillsリポジトリが活発です。

注目例：
- **alirezarezvani/claude-skills**：235以上のproduction-ready Skills/Plugins。Claude Code、Codex、Gemini CLI、Cursorなど11ツール対応。エンジニアリング、マーケティング、complianceなど幅広いカテゴリ。
- **heilcheng/awesome-agent-skills**：実務チームが作成したAgent Skillsのキュレーションリスト。テキストファイル形式でガイドのように読み込まれ、Claude Code/Codex/Gemini CLIなどで即利用可能。
- **K-Dense-AI/scientific-agent-skills**：133以上の科学・研究向けSkills（がんゲノミクス、分子動力学、データベース統合など）。AIエージェントを「AI Scientist」に変えるものとして注目。
- **zainhas/skills**：Together AI製品（inference、training、embeddingsなど）向け12 Skills。Claude Code/Cursor/Codex/Gemini CLI対応。

これらは「Skills = Markdown指示ファイル」で、エージェントがタスク時に自動ロード。Firecrawl（web scraping/automation）やFrontend Design（生産級UI）、Superpowers（multi-step計画/subagents/TDD）などもClaude Code Skillsとして推奨されています。

Xでは、Claude Code + Gemini CLIの組み合わせや、tmux sidebarで複数coding agentを管理するプロジェクト（opensessions）が言及され、Skillsを活かしたワークフローが共有されています。
