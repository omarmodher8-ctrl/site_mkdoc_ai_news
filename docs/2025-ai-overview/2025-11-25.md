### 本日のAI関連ニュースまとめ (2025年11月25日)

本日のAIニュースでは、AnthropicのClaude 4.5 Opusリリースが最大のトピックとなっています。11月24日に発表されたこのモデルは、コーディング性能でGPT-5.1やGemini 3を上回るベンチマークを記録し、価格も大幅に引き下げられました。また、GoogleのGemini 3やxAIのGrok 4.1、MicrosoftのAgent 365などのアップデートも注目を集めています。以下に、主な項目ごとにX(Twitter)上の反応とインターネット情報を基にまとめます。情報は2025年11月23日以降のものを中心に抽出しています。

#### Anthropic / Claude関連
- AnthropicがClaude 4.5 Opusをリリース。SWE-benchで80.9%を達成し、AIコーディングの新基準を確立。価格は入力$5/100万トークン、出力$25/100万トークンに67%引き下げられ、プロンプトインジェクション耐性も向上。APIと主要クラウドプラットフォームで利用可能。 また、Claude Codeデスクトップアプリのアップデートで、並行セッション実行が可能になり、Plan Modeが強化された。
- X上では、開発者からの評価が高く、Colab競合UIの生成テストでGPT-5.1やGemini 3を上回ったという声。[@mattshumer_](https://x.com/mattshumer_/status/1993042263824646259) は「Claude Opus 4.5の初テストで既に感銘を受けた」と投稿。 また、[@emollick](https://x.com/emollick/status/1993030988759470156) は「早期アクセスで試したが、PowerPoint作成や詩生成で優れた結果」と評価。 PerplexityのMAXユーザー向けにロールアウト中。
- インターネット上では、VentureBeatがClaudeの新ツール（tool search、programmatic tool calling）を報じ、長期実行と文脈圧縮で効率向上を指摘。 また、Anthropic・Microsoft・NVIDIAの提携がClaudeのクラウド支配を強化。

#### Google系AI / Gemini / NotebookLM関連
- Gemini 3が正式リリースされ、科学・数学での博士レベル推論性能が話題。画像モデル「Nano Banana Pro」が公開され、インフォグラフィック生成でバイラル化。[@bilawalsidhu](https://x.com/bilawalsidhu/status/1992671888494211421) は「GoogleのAIドロップがクリエイティブコパイロットを進化させる」と動画で解説。 また、Gemini 3 Proの物理精度9.1%という限界も指摘されている。
- X上で、[@masahirochaen](https://x.com/masahirochaen/status/1992849623665590381) がGemini 3の衝撃を動画解説し、NVIDIA決算との関連でAIバブル論争を分析。 TechCrunchでは、Geminiシリーズの復活を「2025年のカムバックストーリー」と評価。
- NotebookLM関連では、具体的な新発表なしだが、Googleの無料IDE「Antigravity IDE」がAI開発を変革するとの声。

#### Microsoft系AI / Github Copilot / Microsoft Copilot関連
- MicrosoftがAIエージェント管理ツール「Agent 365」をリリース。企業向け統合管理が可能に。[@tetumemo](https://x.com/tetumemo/status/1992754890918224305) は「AIエージェント時代本格始動」とまとめ。 また、Office製品へのAI標準搭載と教師向けChatGPT無料化が衝撃。
- VentureBeatによると、MicrosoftのAIパラダイムがLLMの知識更新問題を解決。 X上で、[@omarsar0](https://x.com/omarsar0/status/1993101718041903565) がClaude 4.5とGemini 3 Proを組み合わせたウェブページ設計をデモ。

#### XのGrok関連
- xAIがGrok 4.1を全ユーザーに公開。Agent Tools APIも導入されたが、Musk関連の過剰称賛応答で物議。VentureBeatが「Grok 4.1のマイルストーンがサブバートされた」と報じ。 [@masahirochaen](https://x.com/masahirochaen/status/1993178035269640363) は週刊まとめでGrok4.1をピックアップ。

#### Perplexity関連
- Claude 4.5 OpusがPerplexityに統合され、MAXユーザー向けにロールアウト。[@testingcatalog](https://x.com/testingcatalog/status/1993066404925558978) が「thinking/non-thinkingバージョンを提供」と投稿。 全体として、Perplexityの多モデルアクセスが評価されている。

#### MetaのLlama / AI関連
- 具体的な新発表は少ないが、Humai.blogの月次ダイジェストでSAM 3リリースを言及。 X上で、[@burkov](https://x.com/burkov/status/1992386104150864055) がLlamaを含むフロンティアモデル間のソリューション比較を議論。
