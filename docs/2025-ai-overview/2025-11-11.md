### 2025年11月11日のAI関連ニュースまとめ

本日（2025年11月11日）のAI関連ニュースを、X（旧Twitter）上の投稿やインターネット上の情報から集めました。焦点を指定された項目に絞り、今日の新着情報を中心にまとめます。古い情報は排除しています。

#### OpenAI / ChatGPT関連
- OpenAIのCEOであるサム・アルトマン氏が、Slackのようなツールが「偽の仕事」を生み出していると批判。AIネイティブなオフィスツールの必要性を強調し、ドキュメントやメールを置き換えるエージェントベースのシステムを提唱。既存ツールの置き換えが今後の波になるとの見解が注目を集めている。  
  [@masahirochaen](https://x.com/masahirochaen/status/1988075112827154896) の投稿では、この発言を基にした動画解説が共有され、CapCutの音声変換機能を使ったリップシンク動画が話題に。

#### Claude / Anthropic関連
- AnthropicのClaudeが小説投稿サイトの校正機能として活用され、一部の反AI派がこれを「生成AI」ではなく「解析AI」と独自解釈して許容する姿勢を示している。AIの分類議論が再燃。  
  [@super_bonochin](https://x.com/super_bonochin/status/1988107904503943168) がこの状況を指摘し、データセットに関する議論の矛盾を批判。  
- Claude Code on the Webの無料クレジットが残り1週間程度で終了。活用方法を検討するユーザーが増加中。  
  [@hAru_mAki_ch](https://x.com/hAru_mAki_ch/status/1988112278223564942) が終了間近を報告。

#### Google系AI / Gemini / Jules / NotebookLM関連
- GoogleがGemini File Search Toolを導入。これはフルマネージドのRAG（Retrieval-Augmented Generation）システムで、ファイルアップロード後すぐにクエリ可能で料金も安価。RAGを簡易化するツールとして注目。  
  Redditのr/GeminiAIサブレッドで本日報告され、NotebookLMとの連携も期待されている（）。  
  [@currypurin](https://x.com/currypurin/status/1988092667730022801) がYouTube動画で使い方を解説し、Difyとの比較を述べている。  
- JulesのTools CLIが大幅アップデート。並列タスク実行、TUIでのサイドバイサイド差分表示、自動リポジトリ推論、WSL/Arch Linuxの認証修正を追加。npm経由でインストール可能。  
  [@julesagent](https://x.com/julesagent/status/1988036906337398815) が更新を発表。

#### Microsoft系AI / GitHub Copilot / Microsoft Copilot関連
- Microsoft Copilotのエージェント機能が実行可能になり、カスタムエージェント作成が簡単。ギャル風指示などの遊び心あるテストが話題。  
  [@hAru_mAki_ch](https://x.com/hAru_mAki_ch/status/1988102909763203391) が実行動画を共有。  
- GitHubがAI統合プラットフォーム「Agent HQ」を発表。開発ワークフローを一元化し、エージェント・ワークフローをGitHub Actionsに組み込む動きが進む。マルチエージェント路線が推奨される。  
  [@kamui_qai](https://x.com/kamui_qai/status/1988035276623802593) がZDNET記事を引用し、発表を報告。  
  [@kamui_qai](https://x.com/kamui_qai/status/1988037359557111890) はGitHub ActionsとAIの統合を強く推奨。
