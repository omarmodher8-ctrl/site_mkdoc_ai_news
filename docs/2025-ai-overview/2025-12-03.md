### OpenAI / ChatGPT 関連
OpenAIは、GoogleのGemini 3.0の影響で「コードレッド」（緊急事態）を宣言し、ChatGPTのユーザー数が減少したため、広告導入や新機能（買い物・健康エージェント、パーソナライズレポート）の開発を延期。代わりにチャット機能の改善、画像生成AIの強化、応答速度向上に集中している。内部では「Shallotpeat」と「Garlic」という新モデルを開発中であり、来週にはGemini 3.0を超える長考推論モデルをリリース予定。[@ImAI_Eruel](https://x.com/ImAI_Eruel/status/1996067752755523770)によると、これによりOpenAIは全方位的な開発を進めているが、競合のAnthropicやDeepSeekに劣勢を強いられている可能性がある。

また、[@masahirochaen](https://x.com/masahirochaen/status/1996022449935405196)は、OpenAIのアルトマンCEOがコードレッドを発令した背景を指摘し、個人ユーザーではChatGPTの定着が強いが、法人シェアの争いが鍵になると分析している。[@kinopee_ai](https://x.com/kinopee_ai/status/1996086826835321036)は、OpenAIがChatGPT改善にフォーカスし、Anthropicがエージェント強化を進める対比を述べている。

### Claude / Anthropic 関連
AnthropicはClaude Agent SDKをリリース（旧Claude Code SDK）。これによりエージェント構築が容易になり、コーディングやエンタープライズに特化した強化が進む。[@akira_papa_IT](https://x.com/akira_papa_IT/status/1996024082589012424)は、このSDKの概要を説明し、ツールキットとしての利便性を強調している。
