ここ一週間（2026年2月1日以降、特に2月3日〜8日頃の直近情報）に焦点を当て、**Model Context Protocol (MCP)** に関するAI活用のニュース・議論をX（旧Twitter）とウェブからまとめます。MCPはAnthropicが推進するオープン標準プロトコルで、AI（特にClaudeなど）が外部データ、ツール、サービスに安全・標準的に接続するための「AIのUSB-C」と呼ばれるもので、最近の採用拡大が目立ちます。3日以上前の情報は除外しています。

### 主な直近トピック
1. **GoogleのDeveloper Knowledge APIとMCPサーバーのパブリックプレビュー発表**  
   Googleが公式デベロッパードキュメントへの機械可読アクセスを標準化するDeveloper Knowledge APIを発表し、関連するMCPサーバーを公開プレビュー開始。これにより、AIエージェントがGoogleのドキュメントを効率的に参照可能に。  
   X上で@aktsmmが日本語で共有（4日前の記事参照）: Googleのドキュメント検索がMCPで強化され、標準化ゲートウェイを実現。  
   また、@19nishantが「これはAIのUSB-Cの瞬間」と評価し、Serenities AIでの実践経験を基に、ゼロハルシネーションでのデータ接続のゲームチェンジャーと指摘。

2. **政府・連邦機関でのMCP採用（公的データアクセス向上）**  
   米国連邦政府がMCPを標準として採用し、ChatGPTなどのサードパーティAIチャットボットが公的データを正確に参照可能に。FedScoopが報じ、クエリが公共データで強化される点を強調。  
   これにより、AIの正確性と信頼性が政府業務で向上する動き。

3. **Amazonのagentic広告でのMCP採用**  
   Amazonがagentic広告（AIエージェント駆動の広告）でMCPを活用。エコシステムでの大量採用を示唆し、商用展開の加速を表す。

4. **セキュリティ・実践面の議論（サンドボックス化など）**  
   @mrlatte_storyがAIエージェントのシェルアクセスデモ（12秒で突破）を挙げ、MCPサーバー経由の供給網攻撃リスクを指摘。Dockerコンテナによるランタイム隔離を現実的な防御策として推奨。LLM生成コード実行時のサンドボックスアーキテクチャの重要性を強調。

5. **ツール・製品統合の事例増加**  
   - Whale CLIがMCPサーバーサポートを発表（/mcp-configで外部AIサービス設定）。  
   - InflectivデータプラットフォームがClaude/ChatGPT/GeminiへのMCP対応で、独自データをAIに直接供給可能に。  
   - SarvaData MCP（@TechByAnk）がrawデータセットとエージェント推論のセキュアブリッジとしてGitHubリポジトリ公開。  
   - Salesforce記録をリアルタイムで引き出すデモ（AnthropicのMCPでカスタムコード不要）。

6. **技術コミュニティ・記事の動向**  
   技術サイトではMCPサーバー構築ガイドやセキュリティ解説が活発。  
   - Medium上で「MCP Server Explained」（初心者向けクライアント-サーバーアーキテクチャ解説）。  
   - 「MCP Security 101」（エージェントAI向けの認証・ポリシー適用焦点）。  
   - DEV CommunityやMediumでインタオペラビリティ解決やテスト自動化向けMCPサーバー特集（例: QAツール接続）。  
   これらはMCPを「AIエージェントの標準接続層」として位置づけ、企業導入加速を予測。
