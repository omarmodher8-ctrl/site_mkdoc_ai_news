### AIに使われるMCP (Model Context Protocol) の直近ニュースまとめ（2025年12月4日〜12月7日）

ここ一週間のMCP関連ニュースを調査した結果、12月4日以降の新鮮なトピックを中心に、AIエージェントのセキュリティ強化、ブロックチェーン統合、開発ツールの進化、プロトコルの更新が目立っています。以下にX (Twitter) のユーザー発言、インターネット上のニュース、技術サイトの投稿を基にまとめます。全体として、MCPはAIの外部ツール連携を標準化する動きが加速しており、企業や開発コミュニティでの採用が進んでいます。

#### 主要ニュースの概要
- **MCPのセキュリティ強化と脆弱性対応**: MCPサーバーのセキュリティリスクが注目を集めています。例えば、Unit 42の研究で、MCPのサンプリング機能が悪用されやすいプロンプトインジェクション攻撃の新手法が発見されました。これにより、コマンド実行や会話ハイジャックが可能になるため、レート制限やユーザー承認の強化が推奨されています。[@YusukeNakajima](https://x.com/nakajimeeee/status/1997137342604402916) は、この攻撃の詳細を日本語で解説し、防御策を提案しています。
  
- **ブロックチェーンとAIの融合**: CovalentがMCPを活用したAIエージェントSDKを発表。ブロックチェーン上のリアルタイムデータをAIモデルにセキュアに提供可能になり、dAppsの開発を加速します。[@Arjun | Web3](https://x.com/KoranaArjun126/status/1997337402805997701) がこの統合の利点を共有し、暗号通貨のデータ検証を強調しています。

- **開発ツールのMCP対応拡大**: GitLab Duo Agent PlatformがMCPをサポートし、AIツールが内部システムにアクセス可能に。VS CodeやClaude Desktopなどの外部AIがGitLabインスタンスに接続できます。また、Floorpブラウザのバージョン12.8.0でMCPサーバー対応が追加され、ブラウザ操作をAIツールで可能に。[@🦊 Floorp Web Browser ଳ](https://x.com/Floorp_Browser/status/1997189945644273956) がリリースを発表しています。

- **プロトコル更新とコミュニティ活動**: AnthropicがMCPを更新し、曖昧なクエリ処理を改善。[@てっくたいむ君@色々やってます](https://x.com/tech_time555/status/1997108752030617815) がこれを指摘し、推薦システムでのパーソナライズ強化を議論。MCP Developers Summitでプロキシやレジストリの進捗が共有され、[@MCP Developers Summit](https://x.com/mcpsummit/status/1996641089667928559) がライブイベントを開催。

- **企業向けアプリケーションの進化**: SAP CommunityでMCPを活用したエージェントワークフローが議論され、SAP BTP Kymaとの統合でAIシステム構築を容易に。[@INOAUG](https://x.com/inoaug/status/1997234567380627641) がMCP for APEXの利点を強調しています。
