### 本日のAI関連ニュースまとめ (2025年8月26日)

本日のAIニュースを、X(Twitter)上の投稿（特に@testingcatalogのアカウントを重視）やインターネット検索結果から集約しました。対象項目を中心に、今日の日付に限定した新鮮な情報を日本語でまとめます。主なトピックはOpenAI/ChatGPT、Claude/Anthropic、Google/Gemini、Microsoft/Copilot、XのGrok、MetaのLlama、DeepSeek、その他の有力AIモデル/リサーチ、AIエディタ/CLI、AIブラウザ/資料作成ツールです。セキュリティ問題や市場動向も含めます。

#### OpenAI / ChatGPT
- OpenAIのCEOであるSam Altman氏が、GPT-5のロールアウトでミスを犯したことを認め、有料ユーザー向けにGPT-4oの「温かみ」を復元したと発表。APIトラフィックが48時間で2倍になり、GPU不足が深刻化している。週間のリーチは7億人に達し、データセンターに数兆ドルを投資する計画を明らかにした。また、脳-コンピュータインターフェースのスタートアップを支援し、AIネイティブのソーシャルアプリを探求中。
- ChatGPTが生成AI収益の63%を占め、H1 2025で約19億ドルの市場をリード。アジアでの成長が顕著で、北米が収益の中心。
- Xの投稿では、ChatGPTがコードを台無しにしやすく、Grok 4がそれを修正する例が共有され、AIの黄金時代を皮肉る声も。 また、ChatGPTを個人的なカジュアル質問に最適とする意見。
- セキュリティ問題: Microsoft Copilotのエージェントポリシーバイパスにより不正アクセスが可能。 また、xAIがAppleとOpenAIを提訴し、App Storeでの優遇がGrokに不利と主張。

#### Claude / Anthropic
- AnthropicのCEO、Dario Amodei氏が、AIがエントリーレベルのホワイトカラー職の半分を消滅させ、失業率を20%に押し上げる可能性を警告。
- Claude 4.1が執筆とメモリ管理に最適で、論理・推論に強いと評価。 コード生成に特化し、Claude CodeのDNSデータ流出脆弱性 (CVE-2025-55284) が報告。
- X投稿では、Claudeがコード作業に推奨され、Claude 4がフロンティアモデルとしてリストアップ。 また、LLMのイデオロギー偏向調査で、Claudeを含むモデルが左寄りのシンクタンクを好む傾向。

#### Google系AI / Gemini
- @testingcatalogがGoogleのLogan Kilpatrick氏の「Gemini」投稿に「まさに待っていたもの」と反応し、期待を強調。 また、Demis Hassabis氏（Google DeepMind CEO）の顕微鏡下の奇妙な物体投稿を引用し、24時間以内の発表を示唆（バナナ？）。
- AppleがGoogleのGeminiをSiri強化に検討中。プライバシーを維持しつつAppleサーバーで運用。 US政府がGemini for Governmentを低価格（1機関あたり0.47ドル）で大量導入、市場持続性に疑問。
- GoogleがGeminiを全製品エコシステムに急速統合。Agent Mode（自律探索）、Gemini Go（共同アイデア生成）、Immersive View（視覚回答）の実験モードを導入。 Gemini 2.5が研究・画像生成に最適。
- 人気チャットボット調査: Geminiが17億回のWeb訪問で3位。 APIや長文文書分析に強い。

#### Microsoft系AI / GitHub Copilot / Microsoft Copilot
- GitHub Copilot for AzureがVisual Studio 2022でローンチ。Azure開発ツールをゼロセットアップで統合。 Visual Studio 2022がBYO Modelサポートを追加、Copilot Chatに独自モデルを導入可能。
- Microsoft ExcelにCopilot導入、自然言語で数式入力可能だが、財務報告など高精度タスクには警告。 Copilotが9億5,720万回のWeb訪問で6位。
- セキュリティ問題: Sage Copilotが顧客アカウント間で財務データを漏洩。

#### XのGrok
- Grok 4がコード修正に優れ、ChatGPTの失敗を補う例が共有。Grok 3が複雑問題・研究に最適。 Grokが6億8,690万回のWeb訪問で7位。
- xAIがAppleとOpenAIを提訴、App Store優遇がGrokに不利と主張。 Grok 4がフロンティアモデルとしてリストアップ (7月9日リリース)。

#### MetaのLlama
- MetaがOpenAIのトップ研究者を雇い、LlamaのAI推論を推進。オープンソースと実問題解決に影響大。 Llama 4 Scoutが4月5日リリースのフロンティアモデル。 Meta AIが1億3,040万回のWeb訪問で9位。

#### DeepSeek
- DeepSeek V3.1が8月21日リリースの最新フロンティアモデル。 オープンソースパワーに強く、27億回のWeb訪問で2位。

#### その他の有力AIモデル / AIリサーチ
- NvidiaがJetson Thorを発表: Blackwell搭載のロボット用スーパーブレイン、AmazonやMetaなどで採用。物理AIの実現に寄与。
- Sentient AGIのGRIDエコシステム: 100以上のパートナーと統合、分散コンピュートとデータソースでオープンAGIを目指す。パートナー例: CrunchDAO、PerleLabs、0G_labsなど。
- Vibe codingが開発を変革する一方、AI生成コードの技術的負債と脆弱性が警告。

#### AI色が強いエディタ / CLI
- Cursorがペアプログラミングに最適。 Gemini CLIが開発者向けにローンチ、端末内でコーディング・問題解決を支援 (Googleの7月発表だが、今日の文脈で言及)。

#### Genspark / DIA / Manus / Skywork / Gamma などのAIブラウザ / 資料作成
- Gammaが生産性ツールとしてリストアップ、プレゼンテーション作成に特化。 Perplexityが収益共有を約束、Comet Plusプランでメディアに80%還元。AI検索の倫理的エコシステムを推進。 (他の特定ツールの今日ニュースは少ないが、GammaはAI資料作成の代表例として言及。)

全体として、AIの統合が進む一方、セキュリティと雇用の懸念が目立つ。@testingcatalogの投稿はGemini関連の興奮を強調し、DeepMindの潜在的な発表を予感させる。
