### VSCode (Visual Studio Code)
VSCodeは引き続き安定した人気を保ち、Microsoftの公式アカウントから積極的なアップデート告知が見られます。特に2026年の初回ライブリリースイベントが注目を集めています。

- **@code** が2026年最初のVS Code Live Release Highlightsを1月22日に開催すると告知。最新機能のデモが行われました。[@code](https://x.com/code/status/2013750394623565937)
- 同アカウントからUIトリックのショート動画（8つのTips）を共有し、多くのエンゲージメントを集めています。[@code](https://x.com/code/status/2012802724241527053)
- 最もダウンロードされた拡張機能に関するショート動画も人気で、コミュニティの関心の高さを示しています。[@code](https://x.com/code/status/2012976189879587117)
- GitHub Copilot SDK関連の週末ハッカソンコンテストをプロモートする投稿もあり、AI統合のエコシステムが活発です。

全体として、VSCodeは「退屈だが勝ち続ける」存在として、AIフォーク（CursorやAntigravity）と比較されつつも基盤としての強さを維持しています。

### Cursor
CursorはAIエージェント機能の強化が目立ち、@cursor_aiからの連続したアップデート告知が続いています。バージョン2.4を中心にsubagentsやskillsの導入が話題です。

- **@cursor_ai** がAgent Skillsのリリースを発表。エージェントが専門プロンプトやコードを自動発見・実行可能になり、コンテキスト効率が向上。[@cursor_ai](https://x.com/cursor_ai/status/2014753596223770841)
- Subagentsによる並列タスク実行、画像生成、clarifying questions機能の追加をデモ動画で紹介。全体実行が高速化し、長時間タスク対応が強化されました。[@cursor_ai](https://x.com/cursor_ai/status/2014433672401977382)
- Dynamic context discoveryの詳細ブログを共有し、ファイルシステム活用によるトークン削減（最大46.9%）を強調。[@cursor_ai](https://x.com/cursor_ai/status/2014753598727696717) のスレッド関連。

CursorはVSCodeフォークとして移行しやすく、AIペアプログラミングの進化が速く、開発者から高い評価を受けています。

### Antigravity
Antigravity（Google DeepMind関連のagent-first IDE）はリリース直後から議論が活発ですが、ここ一週間ではアップデートやユーザー体験談が散見されます。VSCodeベースのフォークとして注目されつつ、制限や比較の声も。

- Googleの公式関連で、DeepMind CTOがAntigravityをGoogleのAIモデル開発支援に活用している議論を動画で共有。agentic開発の未来を強調。 
- ユーザーからは「VSCodeやCursorから乗り換えて満足」「multimodal debuggingが強力」といった肯定的意見。一方で、ClaudeリミットやGeminiの品質低下を指摘する声も。
- オープンソースのAgentKit（governance層追加）がAntigravity向けにリリースされ、meta-agentsや依存チェックの自動化を可能に。
- ウェブニュースでは、agent-firstアーキテクチャやGemini 3統合が生産性向上の鍵として取り上げられ、VSCode置き換えの可能性を議論する記事が複数。

情報が新しく変動しやすいですが、agent中心の革新としてVSCode/Cursorとの比較で話題沸騰中です。
