### Anthropic / Claude関連の大きな動き
Anthropicが最上位モデル**Claude Opus 4.6**をリリース。100万トークンのコンテキストウィンドウ（ベータ）に対応し、エージェントチーム機能や高度なコーディング・計画能力が強化。Terminal-Bench 2.0で65.4%の精度を記録し、競合を上回る性能を示した。Claude Codeの勢いをさらに加速させるアップデートで、無料クレジット配布も実施中。

- [@sama](https://x.com/sama) や関連コミュニティで即座に反応が見られ、OpenAI側もこれに対抗する動きが活発化。
- メディアでは[@TechCrunch](https://techcrunch.com/2026/02/05/anthropic-releases-opus-4-6-with-new-agent-teams) などで「agent teams」機能が注目され、生産性向上に直結すると評価。

### OpenAI / ChatGPT / Codex関連
OpenAIがAnthropicのリリース直後に**GPT-5.3 Codex**を投入。Codex（エージェント型コーディングツール）の能力を強化する専用モデルで、複数エージェントの並行開発を加速。企業向けプラットフォーム**Frontier**も発表され、AIを「co-worker」として管理・運用する基盤を提供。CodexアプリでGitワークツリーを使った衝突回避やオートメーションが可能に。

- これにより、Claude Code vs. Codexの競争が激化。[@OpenAIDevs](https://x.com/OpenAIDevs) 周辺で開発者からの実践報告が増加。
- [@ArsTechnica](https://arstechnica.com/information-technology/2026/02/ai-companies-want-you-to-stop-chatting-with-bots-and-start-managing-them) では「チャットから管理へ」のトレンドとして両社の動きを分析。

### Perplexity関連
Perplexityが**Deep Research**を「Advanced」版にアップグレード。外部ベンチマークでSOTA（State-of-the-Art）を主張し、DRACOというオープンソースベンチマークも公開。決定支援分野で強いパフォーマンスを発揮。

- [@perplexity_ai](https://x.com/perplexity_ai/status/2019126571521761450) の公式発表がX上で拡散され、研究用途での信頼性向上として話題。
- Latent Spaceのまとめでも、Perplexityの動きが資金調達ラッシュ（ElevenLabs、Cerebras）と並んでピックアップ。

### Google / Gemini関連
GoogleがChromeサイドパネルに**Gemini 3**を統合した新体験を展開。Nano Banana統合などブラウザワークフローとの密接な連携を強化。

- [@GoogleAIStudio](https://x.com/GoogleAIStudio) や関連アカウントでUX変更の詳細共有。
- トラフィックシェアではGeminiが20%超えを達成し、ChatGPTのシェア低下（65%未満）と対照的に勢いを増している報告あり。
