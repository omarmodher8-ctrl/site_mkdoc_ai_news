本日のAI関連ニュース（2026年2月7日時点で、過去2日以内の最新情報）を、X（旧Twitter）上の関連アカウント投稿やウェブ検索からまとめました。主にOpenAI/ChatGPT、Anthropic/Claude、Google Gemini、xAI/Grok、Perplexity、Meta Llama、DeepSeekなどの有力モデルやツールに焦点を当て、古い情報は除外しています。

### OpenAI / ChatGPT 関連
OpenAIがAnthropicの新モデルリリース直後に、agentic coding特化の新モデル**GPT-5.3 Codex**を発表しました。コーディングを超えた高速処理と多機能性が特徴で、開発者向けにCodexツールを強化。Anthropicとの競争が激化しており、両社のライバル関係がオンライン上で公に衝突する場面も見られます。

X上では、[@OpenAIDevs](https://x.com/OpenAIDevs) や [@sama](https://x.com/sama) 関連の議論が活発ですが、具体的な本日投稿は市場シェア変動の文脈で言及が多く、ChatGPTのシェアが低下傾向（64-65%前後）ながら依然トップを維持している点が話題です。

また、AppleのCarPlayがChatGPTなどの外部AIチャットボットをサポートする方向で開発中との報道もあり、車載AI統合が進む兆しが見えます。

### Anthropic / Claude 関連
Anthropicが**Claude Opus 4.6**をリリース。100万トークンのコンテキストウィンドウ、agent teams（エージェントのチーム連携）、agent swarmsなどの新機能で、複雑なマルチステップタスクやagentic coding、金融分析で強みを発揮。生産品質に近い初回出力が可能と評価されています。

これにより、MercorのAIエージェントベンチマークでスコアが急上昇（one-shotで約30%、複数試行で45%）。Super Bowl広告で「広告なし」を強調し、OpenAIの広告テストを暗に批判する動きも。

Xでは[@claudeai](https://x.com/claudeai) 周辺でagent機能のデモ投稿が散見され、開発者からの注目が高いです。

### Google / Gemini / NotebookLM 関連
Geminiの市場シェアが急伸（15-21.5%程度）し、ChatGPTの独占を崩す形で話題。Gemini 2.0シリーズのFlash版が正式展開され、日常タスクでの高速応答とベンチマーク向上をアピール。

NotebookLM関連の新規投稿は本日目立たず、全体としてGoogleのエコシステム連携（カレンダー/メールなど）がビジネス利用で評価されています。

### xAI / Grok 関連
Grokのシェアが3%超え、DeepSeekに迫る勢い。Grok-3のリリース後評価が高く、Chatbot Arenaで1400超えスコアを記録し、数学・コーディング・推論でChatGPT/Gemini/DeepSeekを上回る盲テスト結果が話題継続中。

[@xai](https://x.com/xai) や関連アカウント（例: @elonmusk系）でBig Brainモードや自社ファクトチェック機能が引き続き議論されていますが、本日新規の爆発的投稿はシェア変動報告が中心。

### Perplexity 関連
PerplexityのDeep Research機能が無料開放され、競合（OpenAI/Google）の有料ツールに対抗。ベンチマークでHumanity’s Last Examで21.1%スコアと好成績だが、OpenAIの26.6%には及ばず、アクセシビリティで差別化。

市場シェア5.8%前後で成長中、$200M追加資金調達で評価額$20B到達の報道が最近ですが、本日直接的新規ニュースは少ない。

### その他有力モデル・ツール
- **DeepSeek**：シェア0.2-3.7%で急成長、特に中国などで強い。Grokに次ぐ上昇傾向。
- **市場全体トレンド**：First Page Sageの2月データでChatGPT 64%、Gemini 15%、Copilot 13.2%、Perplexity 5.8%、Grok 0.6%など。ChatGPTの低下とGemini/Grok/DeepSeekの上昇が顕著。

X指定アカウント（@testingcatalog、@tetumemo、@Shin_Engineer、@masahirochaen、@snakajimaなど）では、これらのモデル比較やシェア変動、agenticツールの実用投稿が散見され、特にClaude/OpenAIの新モデル対決が注目を集めています。全体として、agentic（自律行動型）AIとコーディング/研究ツールの進化が本日のホットトピックです。
