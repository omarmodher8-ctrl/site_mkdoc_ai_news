**本日のAI関連ニュースまとめ（2026年3月26日時点、3月25日以降の最新情報中心）**

### Anthropic / Claude系
- **Claude Codeの日本語入力が大幅改善**：これまで日本語（IME）入力時にカーソル位置がずれる問題が修正され、通常の位置で入力可能に。音声入力依存だったユーザーから「地味だけど最高」と好評です。Claude CodeはAI色強いCLI/エディタとしてさらに使いやすくなりました。
- AccentureとAnthropicが共同で**Cyber.AI**を発表（Claude活用）。セキュリティ運用をAIエージェントで常時稼働化し、リアルタイム対応を強化。
- Claudeのコンピュータアクセス機能がモバイルからも利用可能に（macOSデスクトップ操作をスマホから制御）。
- Anthropicの経済レポートで**AIスキルギャップ**が指摘され、パワーユーザーと新規ユーザーの生産性格差が拡大中。企業はAI実験予算を増やす必要があるとの声が高まっています。

### Google系（Gemini / その他）
- **TurboQuant**アルゴリズムを発表：LLMのメモリ使用量を最大6倍圧縮、処理速度を8倍向上。ローカルデバイス（Mac Miniやスマホ）での高性能AI実行を可能にし、コストも大幅削減。
- **Lyria 3 Pro**音楽生成モデルをリリース。クリエイティブ分野でのGemini活用をさらに推進。
- **Vibe Coding XR**をGeminiとXR Blocksで公開：自然言語プロンプトから物理法則対応のWebXRアプリを60秒以内で生成。Android XRヘッドセット向けプロトタイピングが劇的に加速します。
