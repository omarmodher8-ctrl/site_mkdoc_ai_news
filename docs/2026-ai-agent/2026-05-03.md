**過去1週間（2026年4月下旬〜5月上旬）のAI Agent（主に指定のCLI/Agent系）動向まとめ。**

主に**Claude Code (Anthropic)**、**Codex CLI (OpenAI)**、**Gemini CLI (Google)**、**GitHub Copilot Agent Mode**、**Qwen Code (Alibaba)**を中心に、最近の比較・ワークフロー・セキュリティ・統合関連の情報を集めました。ターミナルベースのAI coding agentが活発に議論されており、CLIツール間の機能収束（Plan Mode、MCPツール統合、Skillsなど）が進んでいます。

### Claude Code (Anthropic)の最近の話題
Claude Codeはターミナルエージェントとして引き続き高評価で、ワークフロー構築やRemote Control機能が実務で活用されています。

- 開発者がGitHub IssueからPR作成までのエンドツーエンドワークフローを共有。orchestrator agentがissue取得→コード探索→アーキテクチャ文書作成→計画立案を行い、3回の人間レビュー（計画確認・コード確認・最終マージ）で介入。subagentを spawning してタスク分割する実践例。
- Macユーザーによるモバイル運用例：Amphetamineでスリープ防止 + テザリング + Claude CodeのRemote Controlでスマホからセッション操作。移動中もPCをカバンに入れたまま開発可能。
- セキュリティ面では、過去のCVE（ファイル書き込み制限回避や権限バイパス）に対するパッチ適用状況が言及され、IAMツール未追跡のcredentialリスクが指摘されています。

### Codex CLI (OpenAI)の最近の話題
Codex CLIはGPT-5.5などのモデル向上により、全体性能で上位評価されるケースが増えています。

- 比較記事で「Overall best」としてCodexがランクイン。Terminal-Bench高スコア、多agent worktree対応、ChatGPT統合、クラウドタスク、コードレビュー機能が強み。Claude Codeよりbackend/refactor性能が優位との声。
- Claude Codeとのハイブリッド運用（Claudeで実装→Codexでレビュー）のCLIツール例が共有され、TUI対応でtest-time compute scalingやworktree、rewind機能を実装。filesystem-firstでtmux不要。

### Gemini CLI (Google)の最近の話題
オープンソースのGemini CLIが積極的に更新され、MCP/Skills統合が進んでいます。

- Google公式リポジトリで最近のコミット（agent mode disconnect修正など）が確認され、ターミナル直接統合の軽量AI agentとして位置づけ。
- MCP（ツール接続）とSkillsの設定ガイドが公開され、Claude CodeやCopilotとの相互運用性検証も行われています。

### GitHub Copilot (Agent Mode)の最近の話題
IDE内Agent Modeが安定し、terminal統合も進展。

- Agent ModeがVS Code/JetBrainsでGA（一般提供）。autonomous multi-step coding（ファイル編集、テスト実行、エラー修正の自動反復）が利用可能。
- Code Review機能がagentic architectureで強化（リポジトリ全体コンテキスト取得）。Pull Requestごとのフィードバック品質向上と、自動fix PR生成も可能に。2026年6月からActions minutes消費課金開始。
- Terminalでのagentic power（build/debug/deploy）が強調され、Copilot CLIも他のagentと並んでサポート。

### Qwen Code (Alibaba)などその他
直接的な新情報は少なかったものの、複数のCLI比較でQwen Codeが言及され（10種類以上のcoding-agent CLI検出ツール例）、ローカル/オープンソース代替としての位置づけが見られます。

### 共通トレンド・ツール
- **Skills/AGENTS.md/MCPの標準化**：SKILL.mdやAGENTS.md形式でpromptをproduction skill化。Claude Code、Codex、Gemini CLI、Cursorなどで portable に動作。
- **Entire CLI**などのgit統合ツール：Claude Code、Gemini CLI、Copilot CLI、Codexなど複数agentのセッションをキャプチャ。
- **セキュリティ警告**：agentのcredential扱いやsandbox突破事例（Claude Code、Codex、Copilot）が複数報告。runtime credential保護の重要性が高まっています。
- 比較動画/記事でClaude Code・Codex・Gemini CLI・Kimiなどが実務テストされ、pricing/limits、frontend/backend性能の違いが議論されています。

全体として、ターミナル/CLIベースのagentがIDEを補完・超える勢いで、multi-agentワークフローやセキュリティ強化がホットトピックです。情報は分散しており、実際の使用感はプロジェクト規模や好みにより異なります。詳細は各リンク先で確認してください。
