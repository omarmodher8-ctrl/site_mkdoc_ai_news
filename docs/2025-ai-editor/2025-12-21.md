### VSCode（Visual Studio Code）のここ一週間の情報

VSCodeの11月2025更新（バージョン1.107）が12月10日にリリースされ、主にAIエージェントの強化とセキュリティ機能の改善が注目された。具体的には、Agent Skillsのサポート開始やPrivate Marketplaceの一般提供、Gemini 3 Flashの統合が話題に。X上では、開発者コミュニティからのフィードバックが多く見られ、企業向けのセキュリティ強化が評価されている。一方で、IntelliCodeのサポート終了が発表され、移行を検討する声も。古いニュースを除外し、12月14日以降の関連情報を中心にまとめた。

- **主な更新とニュース**:
  - バージョン1.107のリリース: 多エージェント調整機能の拡張、TypeScript 7プレビュー、IntelliCodeの廃止。Copilot ChatのURLコンテキスト追加で、ブラウザ制御が可能に（[TechCrunch](https://techcrunch.com/2025/12/11/vs-code-update-brings-agent-overload-typescript-7-preview-and-the-end-of-intellicode/））。
  - JavaScript/TypeScript Modernizerのプレビュー: AI支援でコードを現代化（[Microsoft Developer Blog](https://developer.microsoft.com/blog/jsts-modernizer-preview), 12月9日）。
  - STM32CubeIDE for VS Codeの更新: ツールチェーン強化とデバッグツールバー追加（[ST Community, 12月16日](https://community.st.com/t5/developer-news/stm32cubeide-for-visual-studio-code-what-s-new-in-december-2025/ba-p/847010)）。

- **X上の関連投稿（指定アカウント中心、多様に分散）**:
  - [@code](https://x.com/code/status/2001727543377039647): Agent Skillsのサポートを発表。Anthropicのオープンスタンダードを採用し、スキルを一度作成すればどこでも使用可能（12月18日）。
  - [@mattbierner](https://x.com/mattbierner/status/2001392865650290946): EmilStenstrom氏のVSCodeとエージェントモードを使ったHTML5パーサーの構築事例を紹介（12月17日）。
  - [@mizchi](https://x.com/mizchi/status/2000899685947691035): Jupyter Notebook（VSCode Notebook API）とREPLの関係性について議論（12月16日）。
  - [@ThePracticalDev](https://x.com/ThePracticalDev/status/2000732663317926065): 開発者のキャリアストーリー動画を共有（12月16日）。

### Cursorエディターのここ一週間の情報

- **主な更新とニュース**:
  - Visual Editorリリース: ドラッグ&ドロップとプロンプトでUI設計。Figmaとの連携強化（[WIRED, 12月11日](https://www.wired.com/story/cursor-launches-pro-design-tools-figma/)）。

### Antigravityのここ一週間の情報
