### VSCode（Visual Studio Code）に関する情報

過去1週間のX（Twitter）とインターネット上の議論では、VSCodeの最新アップデート（バージョン1.107）が主な話題となりました。このアップデートでは、AIエージェントの統合が強化され、Agent HQ機能が追加されて複数のエージェントを管理しやすくなりました。また、バックグラウンドエージェントのサポートやTypeScript 7のプレビュー対応、IntelliCodeの廃止（Copilotへの移行）が注目されています。拡張機能として、Mermaid Graphical Editorがシーケンス図やER図の作成に便利だと評価されており、開発中のシームレスな使用が助かるとの声が複数ありました。ターミナルでのIntelliSense補完やTerraform拡張の改善も指摘されていますが、一部ではリージョナルNAT Gatewayの対応が遅れているとの指摘も。全体として、AI機能の進化が生産性を向上させているようです。

### Kiroに関する情報

Kiroの過去1週間は、CLIのアップデートが主な話題で、Language Server Protocol (LSP)統合によりコード理解が向上し、参照検索や診断が自然言語で可能になりました。re:InventでのLabyrinth coding challengeや、BrexやCascoの活用事例が共有され、仕様駆動開発の利点が強調されています。Kiro Powersのリリースも注目され、専門スキルのパッケージ化が開発を加速すると評価。スタートアップ向けの1年無料クレジットも人気です。一部では、vibe codingとの比較で微妙との声も。

- [@kirodotdev](https://x.com/kirodotdev/status/1999592764829274369): Code more intelligently in the terminal with the Language Server Protocol integration. Kiro CLI now has the same code understanding that powers the IDE. Pre-configured with 7 languages (TypeScript, Rust, Python, Go, Java, Ruby, C/C++), you can find references, navigate definitions, and get diagnostics all through natural language prompts. Just run /code init and ask.
