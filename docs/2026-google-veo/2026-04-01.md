過去24時間（2026年3月31日〜4月1日頃）で、Googleの**Veo 3**関連の主な動きは**Veo 3.1 Lite**のリリースです。これはVeo 3.1シリーズの新モデルとして、コストを大幅に抑えた動画生成オプションとして発表されました。OpenAIのSora関連ニュースの文脈で、Googleが動画生成分野へのコミットメントを強調する形で注目を集めています。

### Veo 3.1 Liteの主な特徴とスペック
- **位置づけ**: Veo 3.1シリーズで最もコスト効率の高い「軽量」モデル。Veo 3.1 Fastの半額以下（価格が半分以下）で、同等の生成速度を実現。
- **対応機能**:
  - テキストto動画（Text-to-Video）と画像to動画（Image-to-Video）。
  - 解像度: 720pおよび1080p（4K非対応）。
  - アスペクト比: 横型（16:9）と縦型（9:16）の両対応。
  - 動画長: 4秒、6秒、8秒。
  - 音声: デフォルトでネイティブ音声生成を含む。
- **価格例**（720pの場合）: 0.05ドル/秒（8秒動画で約0.40ドル）。1080pは0.08ドル/秒。
- **提供場所**: 即時ロールアウト中。Gemini APIの有料ティアおよびGoogle AI Studioで利用可能。開発者向けに高速イテレーションやストーリーボード作成に適した位置づけ。

### 追加の価格調整
Googleは2026年4月7日から**Veo 3.1 Fast**の価格も引き下げます。
- 720p: 0.15ドル/秒 → 0.10ドル/秒（または0.12ドル程度の情報あり）。
- 1080p: 同様に値下げ。
- 4K: 0.35ドル/秒 → 0.30ドル/秒。
これにより、高性能モデルもよりアクセスしやすくなります。Google AI StudioおよびGemini API製品責任者のLogan Kilpatrick氏は「Video’s here to stay（動画はここに残る）」とXでコメントし、Veo 3.1 Lite導入とFastの値下げを歓迎する姿勢を示しました。

### X（Twitter）上の反応（指定アカウント中心）
指定されたアカウント群では、Veo 3.1 Liteに関する直接的な詳細投稿は限定的でしたが、以下のものが確認できました。

- [@AiPhotorealGirl](https://x.com/AiPhotorealGirl/status/2039168795181473954) は、Flow by GoogleでVeo 3.1 Liteを実際に試用。テキストto動画で生成速度が速く、品質に大きな低下を感じず「スナックっぽく扱えそう」と評価。一部モーション（座る瞬間など）に微妙な違和感はあるものの、Fastとの違いは微かで、Grok Imagineへの対抗馬的なイメージを持っています。生成例として短い動画を共有。

他の指定アカウント（@Strength04_Xなど）では過去のVeo 3.1活用例（製品ショットや商業風動画のプロンプト共有）が目立ちましたが、過去24時間内のVeo 3.1 Lite特化投稿は確認されませんでした。一般ユーザーからは「コスト半減でアクセスしやすくなった」「AI動画の価格戦争が加速」「品質とコストのトレードオフをどう考えるか」といった議論が見られます。一部では「Veo 4への期待が高まる」との声も。

### GitHub関連の動き
GitHub上ではVeo 3/3.1を扱ったリポジトリが複数存在しますが、過去24時間内の新規更新やVeo 3.1 Lite特化のものは確認されませんでした。主なものは：
- Google公式関連のノートブック（veo3_reference_to_video.ipynb、veo3_advanced_controls.ipynbなど）で、Vertex AIでのVeo 3活用例。
- サードパーティのラッパーやプロンプトガイド（例: GeminiGenAIのVeo3-AI-Video-API、snubrootのプロンプティングガイド）で、既存のVeo 3統合ツールが中心。

新モデルLiteのAPIドキュメントはGoogle公式（Gemini API / AI Studio）で公開されており、開発者向けに即時アクセス可能です。

全体として、過去24時間の焦点は**Veo 3.1 Lite**の低価格化によるアクセシビリティ向上にあり、Googleが動画生成をより幅広い開発者・クリエイターに広げようとする動きが明確です。品質面での評価はまだ初期段階で、実際の使用感はユーザー次第という声も上がっています。さらなるアップデート（Veo 4など）が期待される状況です。
