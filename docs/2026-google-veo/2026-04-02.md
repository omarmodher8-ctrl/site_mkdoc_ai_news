過去24時間（2026年4月1日〜2日頃）のGoogle「Veo 3」（およびVeo 3.1関連）に関する主な動きは、**Veo 3.1 Lite**の公開・プレビューリリースを中心に、コストを抑えた動画生成モデルのアクセシビリティ向上です。Veo 3本体は2025年5月のGoogle I/Oで発表されたモデルで、オーディオ（音声・効果音）統合や高品質動画生成が特徴ですが、今回はその派生版「Lite」版の低コスト化・高速化が話題となっています。古い発表情報は除外し、最新の言及のみまとめます。

### Veo 3.1 Liteの主な特徴とリリース情報
- Googleが**Veo 3.1 Lite**をGemini API / AI Studio / Vertex AI向けに公開。テキスト-to-動画および画像-to-動画をサポートし、4〜8秒程度の短いクリップ（720p〜1080p）を低コスト・高速で生成可能。
- 従来版より価格を抑え（半額以下との声も）、迅速なイテレーションや高ボリューム利用に最適化。プロンプト制御の強さ、リアルなモーション、内蔵オーディオサポートを維持しつつ、開発者・クリエイター向けのスケーラビリティを重視。
- モバイルワークフローや大量生成に適しており、無料アカウントでも動画生成回数が増えるなどの利便性向上も関連して言及されています。

日本語の情報源では、以下のように報じられています：
- 【ぱそこん相談室】（@pc_sdn）が「Google が『Veo 3.1 Lite』を公開 〜コスト半額以下で使える AI 動画生成モデル〜」と記事を共有。詳細はリンク先で確認を。
